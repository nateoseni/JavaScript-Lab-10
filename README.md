Object-Oriented-Programming
In this project I created an inventory management system in which I created classes and objects for products and perishable products
Applied methods to do things like get total value, format strings, apply discounts, add products to an inventory list, and get the inventory value
The program can take variable names, and then will search the inventory list to find a matching name and then returns the information about the product
Included an index.html file to coherently run on a website and output into the console
